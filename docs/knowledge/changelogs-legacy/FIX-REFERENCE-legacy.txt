🔴 BEFORE (v1.0) - THE PROBLEM
═══════════════════════════════════════════════════════════════

Running with sudo saved files to ROOT's home directory:

  Saved to: /root/system-profile-20260115_142211/  ❌
            └─ You can't access this as regular user!

  $ ls ~/system-profile/

      No such file or directory  ❌


🟢 AFTER (v1.1) - THE FIX
═══════════════════════════════════════════════════════════════

Now saves to YOUR home directory, even when using sudo:

  $ sudo ./system-profiler.sh

            └─ You CAN access this!

  $ ls ~/system-profile/
  system-profile-20260115_142211/  ✓
